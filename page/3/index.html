内部类

##理解
把类A定义在类B内部，类A就被称为内部类

内部类语法格式

public class OuterClass{`

}

内部类访问特点

内部类可以直接访问外部类的成员，包括私有

外部类要想访问内部类的成员，必须创建对象

内部类分类

成员内部类

分类

静态内部类（static，属于外部类本身）
非静态内部类（private）
num                    先查找局部变量
this.num            再查找内部类的成员
Outer.this.name;    最后查找外部类对象

局部内部类

局部内部类访问局部变量，必须加final修饰

匿名内部类

没有名字的内部类
本质是一个匿名子类对象

前提

存在一个类或者接口

格式

new 类名或者接口名() {
        重写方法();
};

使用内部类

在外部类内部使用内部类

在外部类内部使用内部类时，与平常使用普通类没有太大区别
注意：不要在外部类的静态成员中使用非静态内部类，因为静态成员不能访问非静态成员

在外部类以外使用非静态内部类

条件:内部类不能使用private修饰
省略修饰符的内部类：只能被与外部类处于同一包中的其他类所访问
使用protected修饰的内部类：可被与外部类处于同一包中的其他类和外部类的子类所访问
使用public修饰的内部类：可以在任何地方被访问

三、抽象类

有些时候，我们对事物不能用具体的东西来描述，这个时候就应该把事物定义为抽象类。

抽象类的特点：

抽象类或者抽象方法必须用abstract修饰
抽象类中不一定有抽象方法，但是有抽象方法的类一定是抽象类
抽象类不能实例化
可以按照多态的方式实例化。
抽象类的子类
    要么是抽象类
    要么重写抽象类中的所有抽象方法

抽象类的成员特点：

成员变量
    可以是变量，也可以是常量
构造方法
    有。
    不能实例化，有构造方法有什么意义呢?
    用于子类访问父类数据的初始化。
成员方法
    可以是抽象的，也可以是非抽象的

abstract不能和哪些关键字共存?

final: 
    final修饰的方法不能被重写，而abstract修饰的方法要求必须被重写。
private：
    private修饰的方法不能被重写，而abstract修饰的方法要求必须被重写。
static：
    static修饰的方法可以通过类名调用，而abstract修饰的方法是没有方法体的，
    调用一个没有方法体的方法是没有任何意义的，所以不能共存。

抽象类应用举例

//抽象类Animal
abstract class Animal {
	int num = 10;
	final int num2 = 20;
	//抽象类构造不能实例化
	public Animal() {}
	//抽象成员方法
	public abstract void method();

	public abstract void function();
}
//继承应用
class Dog extends Animal  {
	public void show() {
		num = 100;
		System.out.println(num);
		//num2 = 200;
		System.out.println(num2);
	}
	//重写抽象方法
	public void method(){
	System.out.println("Dog");
	}
	public void function() {}
}

class Main {
	public static void main(String[] args) {
		Dog d = new Dog();
		d.show();
		d.method();
	}
}

四、接口

有些时候，不是事物本身具备的功能，我们就考虑使用接口来扩展。
接口不提供任何实现，它体现的是规范和实现分离的设计哲学

接口的特点：

关键字interface

interface 接口名 {
    //各种定义，保证成一种规范
}
接口可实现多继承，即一个接口可有多个直接父接口，获得父接口的所有抽象方法、常量。

类实现接口用关键字implements 
    class 类名 implements 接口名 {}
    接口不能实例化
接口的子类
    抽象类
    或者重写接口中的所有方法

接口的成员特点

成员变量
    只能是常量（public static final）
成员方法
    只能是抽象方法（public abstract）

类与接口的关系

类与类
    继承关系，只能单继承，可以多层继承。
类与接口
    实现关系，可以单实现，也可以多实现。
    还可以在继承一个类的同时实现多个接口。
接口与接口
    继承关系，可以单继承，也可以多继承。

抽象类和接口

共性：
    都不能被实例化，位于继承树的顶端，用于被其他类实现和继承
    都可包含抽象方法，实现接口或继承抽象类的普通子类都必须实现这些抽象方法。
区别：
    抽象类：父抽象类，里面定义的是共性内容。
    接口：父接口，里面定义的是扩展内容。

接口应用样例

//定义接口类
interface Animal {
	//接口会自动指认静态常量分配static和fianl修饰符
	public static final int num = 10;
	public abstract void show();
}
//使用接口
class Dog implements Animal {
	public void show() {}
}

class Main {
	public static void main(String[] args) {
		Dog d = new Dog();
		//d.show();
		System.out.println(d.num);
	}
}

程序输出：

10

重写父类

子类扩展了父类，子类是一个特殊的父类。
方法重写注意事项：
A:父类中私有方法不能被重写
B:子类重写父类方法时，访问权限不能更低

C:父类静态方法，子类也必须通过静态方法进行重写。

class Person {

final关键字

很多时候，我们可能不想让子类修改我的内容。这个时候该怎么半呢?
针对这种情况，java又提供了一个状态修饰符：final。
可以修饰类，成员变量，成员方法。

类：            类被final修饰，说明该类是最终类，不能被继承。
成员变量：    变量被final修饰后，就变成了常量。值不能被修改。
成员方法：    方法不能被子类重写。

多态

多态：同一个对象在不同时刻表现出现的不同状态。
当运行该引用对象时，其方法行为总表现出子类的行为特征，而不是父类的行为特征
出现调用同一方法时呈现出不同的行为特征。

例子：

class Animal {
	public void eat() {
		System.out.println("动物吃饭");
	}
}

数据类型和运算符

1. 基本数据类型

    整型
        byte :  一个byte类型在内存占8位         表数范围： -128 ~ 127
        short : 一个short类型在内存占16位       表数范围： -2^15 ~ 2^15-1
        int :   一个int类型在内存里占32位        表数范围： -2^31 ~ 2^31-1
        long :  一个long类型在内存里占64位     表数范围： -2^63 ~ 2^63-1
    字符型
        char : 一个char类型在内存里占2位。
    浮点型(java默认是double)
        float :  4字节，32位
        double : 8字节，64位
    布尔型
        boolean

2. 基本类型的类型转换

     自动类型转换
     强制类型转换    (和C++类似)

3. 运算符

    算术运算符
        + ，- ，* ，/ ，% ，++ ，-- ，
    赋值运算符
        int a = 7;
    位运算符
        &（与）,|（或）,~（非）,^（异或）,<<(左移)，>>（右移），>>>（无符号右移运算符）
    比较运算符
        < , > , <= , >= , ==
    逻辑运算符
        && , || , ! 
    三目运算符
        (expression) ? if-true-statement : if-false-state;
        例如：String str = 5 > 3 ? "5大于3" ："5小于3"
    运算符的结合性和优先级
        分隔符>单目运算符>强制类型转换运算符>乘法/除法/求余>加减法>移位运算符......

流程控制和数组

1. 流程控制

     if,switch,循环结构(for,while),嵌套循环
     
      continue,break的使用

2. 数组

     静态初始化：type []a = new type[element1,element2,......];
    动态初始化：type []a = type[size];
     数组遍历：for,foreach(临时遍历，无法遍历时对数组元素进行更改)
    多维数组：type [][]a = new type[size][size];
    Java8增强的工具类 : Arrays
        数组转字符串：
        int[] array = new int[]{1, 2, 3};
        System.out.println(Arrays.toString(array)); 
        数组排序：
        Arrays.sort(array);
        Arrays.sort(array,0,2);
        数组复制：
        int []arraycopy = Arrays.copyOf(array);
