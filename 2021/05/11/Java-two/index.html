Java（二）数据类型和运算符

发表于 2021-05-11 更新于 2021-05-13 分类于 Java

数据类型和运算符

1. 基本数据类型

    整型
        byte :  一个byte类型在内存占8位         表数范围： -128 ~ 127
        short : 一个short类型在内存占16位       表数范围： -2^15 ~ 2^15-1
        int :   一个int类型在内存里占32位        表数范围： -2^31 ~ 2^31-1
        long :  一个long类型在内存里占64位     表数范围： -2^63 ~ 2^63-1
    字符型
        char : 一个char类型在内存里占2位。
    浮点型(java默认是double)
        float :  4字节，32位
        double : 8字节，64位
    布尔型
        boolean

2. 基本类型的类型转换

     自动类型转换
     强制类型转换    (和C++类似)

3. 运算符

    算术运算符
        + ，- ，* ，/ ，% ，++ ，-- ，
    赋值运算符
        int a = 7;
    位运算符
        &（与）,|（或）,~（非）,^（异或）,<<(左移)，>>（右移），>>>（无符号右移运算符）
    比较运算符
        < , > , <= , >= , ==
    逻辑运算符
        && , || , ! 
    三目运算符
        (expression) ? if-true-statement : if-false-state;
        例如：String str = 5 > 3 ? "5大于3" ："5小于3"
    运算符的结合性和优先级
        分隔符>单目运算符>强制类型转换运算符>乘法/除法/求余>加减法>移位运算符......

流程控制和数组

1. 流程控制

     if,switch,循环结构(for,while),嵌套循环
     
      continue,break的使用

2. 数组

     静态初始化：type []a = new type[element1,element2,......];
    动态初始化：type []a = type[size];
     数组遍历：for,foreach(临时遍历，无法遍历时对数组元素进行更改)
    多维数组：type [][]a = new type[size][size];
    Java8增强的工具类 : Arrays
        数组转字符串：
        int[] array = new int[]{1, 2, 3};
        System.out.println(Arrays.toString(array)); 
        数组排序：
        Arrays.sort(array);
        Arrays.sort(array,0,2);
        数组复制：
        int []arraycopy = Arrays.copyOf(array);

详情用法：https://blog.csdn.net/zhzh402/article/details/79670509
