内部类

##理解
把类A定义在类B内部，类A就被称为内部类

内部类语法格式

public class OuterClass{

}

内部类访问特点

内部类可以直接访问外部类的成员，包括私有

外部类要想访问内部类的成员，必须创建对象

内部类分类

成员内部类

分类

静态内部类（static，属于外部类本身）
非静态内部类（private）
num                    先查找局部变量
this.num            再查找内部类的成员
Outer.this.name;    最后查找外部类对象

局部内部类

局部内部类访问局部变量，必须加final修饰

匿名内部类

没有名字的内部类
本质是一个匿名子类对象

前提

存在一个类或者接口

格式

new 类名或者接口名() {
        重写方法();
};

使用内部类

在外部类内部使用内部类

在外部类内部使用内部类时，与平常使用普通类没有太大区别
注意：不要在外部类的静态成员中使用非静态内部类，因为静态成员不能访问非静态成员

在外部类以外使用非静态内部类

条件:内部类不能使用private修饰
省略修饰符的内部类：只能被与外部类处于同一包中的其他类所访问
使用protected修饰的内部类：可被与外部类处于同一包中的其他类和外部类的子类所访问
使用public修饰的内部类：可以在任何地方被访问

在外部类以外的地方定义内部类：

OuterClass.InnerClass VarName;
OuterInstance.new InnerConstructor();

例子

class Outer {
    private int num = 10;
    //成员位置
        class Inner    {
            public void show() {
                System.out.println(num);
            }
        }
        public void method() {
            Inner i = new Inner();
            i.show();
        }
    }
    class InnerClassDemo {
        public static void main(String[] args) {
        //错误
        //Inner i = new Inner();
        //格式：外部类名.内部类名 对象名 = 外部类对象.内部类对象;
        Outer.Inner oi = new Outer().new Inner();
        oi.show();
    }
}

例子二(static修饰的内部类外部使用)

class Outer {
        public static class Inner {
            public void show() {
                System.out.println("show");
        }
        public static void show2() {
            System.out.println("show2");
        }
    }
}
class InnerClassDemo2 {
    public static void main(String[] args) {

        //用static修饰后，这种写法错误
        //Outer.Inner oi = new Outer().new Inner();
        //用类名限定
        Outer.Inner oi = new Outer.Inner();
        oi.show();
        oi.show2();
        Outer.Inner.show2();
    }
}
输出
show
show2
show2

例子三（接口和匿名内部类）

interface Product{
    public double getPrice();
    public String getName();
}
public class Main{
    public void text(Product p){
        System.out.println("购买了一个"+p.getName()+"花掉了"+p.getPrice());
    }
    public static void main(String []args){
        Main ta = new Main();
        ta.text(new Product(){
            public String getName(){
                //System.out.print("显卡价格:")
                return "显卡";
            }
            public double getPrice(){
                //System.out.println(105.7);
                return 105.7;
            }
        });
    }
}

例子四（抽象父类和匿名内部类）

abstract class Device{
    private String name;
    public abstract double getPrice();
    public Device() {}
    public Device(String name){
        this.name = name;
    }
    public abstract String getName();
    //public abstract String setName();
}
public class Main{
    public void text(Device d){
        System.out.println("Buy "+d.getName()+",cost "+d.getPrice());
    }
    public static void main(String []args){
        Main ai = new Main();
        ai.text(new Device("kkk"){
               public double getPrice(){
                return 115.7;
                }
               public String getName(){
                   return "ooo";
               }
            });
        Device d = new Device(){ {
            System.out.println("init");
            }
            public double getPrice(){
                return 47.7;
            }
            public String getName(){
                return "keyboard";
            }
        };
        ai.text(d);
    }
}
输出
Buy ooo,cost 115.7
init
Buy keyboard,cost 47.7
