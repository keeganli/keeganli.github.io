一、Java环境配置

下载和安装Java8的JDK（JAVA DEVELOPMENT KIT）

 1. JDK是Java标准开发包，是SUN提供的一套用于JAVA应用程序的开发包，提供了编译，运行JAVA程序所需的各种工具和资源。
 2. JRE是JAVA运行环境，它包含JVM以及其他环境支持，如果只运行JAVA程序，则可仅需要JRE。 
 3. 下载JDK
 4. 配置PATH环境变量

二、第一个JAVA程序

1.说明

因为编译JAVA源代码可以使用任何无格式的文本编辑器，
在Windows上可以使用记事本进行编写，
`win+R    cmd`命令行进行编译运行。

javac 是编译命令

java 是运行命令，注意文件没有后缀
