Java集合框架

在很久之前，Java就提供了各种类，但是这些类之间关联性不大，以至于之间的操作方式差很大，不利于使用

于是便设计了集合框架

集合框架满足的目标

  • 框架高效

  • 允许容纳不同的集合，并且之间具有高度互通性

  • 对于某一集合的扩展是方便和简单的

    框架图如下

集合框架分析

  • 集合框架主要有两种类型的容器，一种是集合(Collection)，存储一个元素的集合；另一种是图(Map)，存储键、值映射
  • 有三种接口类型，分别是List,Set,Queue
  • 具体实现类有很多扩展，如ArrayList、LinkedList、HashSet、HashMap等。
  • 还有各种算法，方便集合的操作
